# Protocolo Think Aloud

## Objetivo
Detectar fricciones reales de navegacion y comprension durante el uso del sistema de examen.

## Tareas
1. Ingresar al sistema e interpretar instrucciones.
2. Responder las preguntas usando teclado.
3. Corregir un envio incompleto.
4. Probar un comando (`limpiar` o `enviar`).
5. Revisar resultado final.

## Guion del moderador
- "Explica en voz alta lo que piensas mientras avanzas."
- "Que esperabas que ocurriera en este paso?"
- "Que parte te parecio confusa o cansada mentalmente?"
